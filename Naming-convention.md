### Size and Spacing

We use T-shirt convention

{% hint style="success" %}
* `S` `M` `L` `XL` `XXL` `XXXL` `XXXXL`
* `S` `M` `L` `XL` `2XL` `3XL` `4XL`
{% endhint %}

{% hint style="danger" %}
* `SM` `MD` `LG` `XL`
{% endhint %}

## Boolean props

{% hint style="success" %}
* `is[whatever]`
* `has[whatever]`
{% endhint %}

{% hint style="danger" %}
* `with[whatever]` Because in React, the prefix `with` is used for HoC
* `use[whatever]` Because in React, the prefix `use` is used for hooks
{% endhint %}
